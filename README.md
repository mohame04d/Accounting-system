# Accounting System

## Description
Accounting System is a desktop application built using **Electron.js**, **Node.js**, and **SQLite** for managing products, inventory, sales, and purchases.  
The system provides an intuitive interface to track products, monitor inventory levels, record sales and purchases, and log all inventory changes efficiently.

## Features
- **Product Management**: Add, update, and track product details including name, price, and description.  
- **Inventory Management**: Monitor stock levels and manage available quantities.  
- **Sales Tracking**: Record every sale with product details and quantities sold.  
- **Purchase Tracking**: Record purchases and update inventory accordingly.  
- **Inventory Log**: Keep a detailed log of all inventory changes, including sales and purchases.  
- **User-Friendly Interface**: Built with **HTML, CSS, and JavaScript** for a responsive desktop experience.  
- **Cross-Platform Desktop App**: Powered by **Electron.js** for standalone desktop usage.

## Technologies Used
- **Electron.js** – for creating cross-platform desktop applications.  
- **Node.js** – for backend logic and database management.  
- **SQLite** – lightweight, file-based database for local storage.  
- **HTML / CSS / JavaScript** – for building the user interface.

## Data Models
- **Product**: Stores information about each product such as name, price, and description.  
- **Inventory**: Tracks product quantities in stock.  
- **Sales**: Records all sales transactions with products and quantities.  
- **Purchase**: Records all purchase transactions and updates inventory.  
- **Inventory Log**: Maintains a history of all inventory changes.
